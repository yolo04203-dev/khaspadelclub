# Khas Padel Club

A competitive padel platform for organizing ladders, tournaments, americano sessions, and team challenges.

## Features

- **Ladders** — Ranked competitive play with challenge-based matchmaking
- **Tournaments** — Single/double elimination and round-robin formats with group stages
- **Americano** — Quick social sessions with automatic partner rotation
- **Teams** — Create and manage doubles partnerships
- **Player Profiles** — Stats, match history, and head-to-head records
- **Admin Panel** — Full control over players, matches, rankings, and permissions

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Capacitor (Android & iOS)
- Framer Motion

## Development

```sh
npm install
npm run dev
```

## Mobile Builds

```sh
npx cap sync
npx cap open android
npx cap open ios
```
